# Changelog

Changes relevant to the users of Cappysan's Asustor Certbot APK are documented here.

This project adheres to [Semantic Versioning](https://semver.org/).

## 5.2.2-r9 - 2026-03-12

### Added

- Logrotate the certbot logs
- Add a reload option to restart only if already started

### Changed

- Made pipx package 'letsencrypt' instead of parent folder
- Moved zip action from deploy to post


## 5.2.2-r5 - 2026-03-02

### Changed

- Improved scripts, including error messages via syslog
- Moved renewal hooks to a standardized path
- Use a specific script to install renewal hooks


## 5.2.2-r3 - 2025-12-20

### Changed

- Changed configuration files to have less of them
- Improved error handling of deploy hooks

## 5.2.2-r2 - 2025-12-19

### Added

- Added 'certbot-apache' plugin support.
- Added 'certbot-dns-cloudflare' plugin support.
- Added 'certbot-dns-digitalocean' plugin support.
- Added 'certbot-dns-gehirn' plugin support.
- Added 'certbot-dns-google' plugin support.
- Added 'certbot-dns-ovh' plugin support.
- Added 'certbot-dns-route53' plugin support.
- Added 'certbot-nginx' plugin support.


### Changed

- Upgraded version to 5.2.2
- Using certmanager is now required to add the certificate

## 4.2.0-r7 - 2025-11-03

### Changed

- Renamed to 'certbot'.
- Improved logs.
- Removed jq requirement.


## 4.2.0-r4 - 2025-11-29

### Changed

- Change renewal-hooks and ordered them.


## 4.2.0-r3 - 2025-11-26

### Added

- Added syslog logging to the application.

## 4.2.0-r1 - 2025-09-23
