# Files

## provider.conf

Set to the provider; one of cloudflare, digitalocean, dnsimple, dnsmadeeasy, gehirn, google, linode, luadns, nsone, ovh, rfc2136, route53, sakuracloud.


## credentials.conf

Add your API credentials in this file, or create a specific file such as `<provider>.conf` (eg: google.conf)`


## domains.conf

A comma separated list with no spaces of domains to validate (eg: *.example.com)
